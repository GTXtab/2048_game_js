## 2048 Game (JavaScript)

A sleek, responsive **2048 clone** built with vanilla JavaScript, HTML5, and CSS3. Experience the classic tile-merging puzzle with smooth animations and a mobile-friendly design.

### Key Features

- **Pure JavaScript:** No heavy frameworks—lightweight and fast.
- **Responsive Design:** Fully playable on desktop and mobile browsers.
- **Smooth Transitions:** CSS3 animations for tile sliding and merging.
- **Local Storage:** Automatically saves your high score and current game state.
- **Touch Support:** Swipe gestures implemented for mobile devices.

### How to Play

Use your **arrow keys** or **swipe** to move the tiles. When two tiles with the same number touch, they **merge into one!** Reach the **2048** tile to win.
